# Anomaly Detection Project

This project implements a machine learning pipeline to identify rare anomalies within a high-imbalance dataset using XGBoost and Neural Networks. In this framework, **Target 1** represents an anomaly, and **Target 0** represents a normal instance.


## 🚀 Setup and Installation

Follow these steps to set up the project using the provided virtual environment configuration:

### 1. Open the Project Folder in Terminal
### 2. Create a Virtual Environment

#### macOS / Linux
```bash

python -m venv venv

source venv/bin/activate

```

#### Windows
```bash

python -m venv venv

venv\Scripts\activate

```

### 3. Install Dependencies
```bash

pip install -r requirements.txt

```

## 4. Run the Notebook File in some code editor (VS Code)
